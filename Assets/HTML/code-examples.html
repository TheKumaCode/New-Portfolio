Form Validation

Explanation

This code takes the email from the input once the submit button is clicked and passes it through a regex, if it doesn't pass the regex then it stops the form from submitting and gives the input a class that shows the user that there is an error.

A regex is an expression that checks over a string under a specific set of rules, rules that include characters contained, in what order they are contained, how many of them are in a row, etc.

If the regex passes it removes the class that shows the user that there is an error, and submits the form.

Explanation

Database

This queries objective is to get all the movies that are under the genre Adventure.

SELECT mov_title AS "Movie Title", mov_year AS "Release Year" FROM movie

This line of code, selects the columns mov_title and mov_year from the movie table in the database, and gives each of them an alias ("Movie Title" and "Release Year" respectively), for easier readability. Selecting a column means that it will be displayed in the output of the query.

WHERE mov_id IN (SELECT mov_id FROM movie_genres .....

This line of code is a subquery to the previous line of code, which further narrows the search down. It does this by clarifying that the value of mov_id in the previous line has to be the same as the value in the subquery. In this case the same as the value of mov_id in the table movie_genres.

WHERE gen_id IN (SELECT gen_id FROM genres WHERE gen_title = 'Adventure')

This is the subquery that is placed within the previous subquery to even further narrow the search, since the value is unknown by the previous subquery. This subquery clarifies to the previous subquery that the value of gen_id needs the be that same as the value of gen_id in the table genres where value attached to the the gen_title equals to the text "Adventure".

ORDER BY mov_year DESC;

This line of code orders the output of the query in descending order of the value mov_year. Meaning from the newest to the oldest.

SQL


        SELECT mov_title AS "Movie Title", mov_year AS "Release Year" FROM movie
        WHERE mov_id IN 
            (SELECT mov_id FROM movie_genres 
            WHERE gen_id IN 
                (SELECT gen_id FROM genres 
                WHERE gen_title = 'Adventure')
            ) 
        ORDER BY mov_year DESC;
